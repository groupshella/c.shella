# شلة (Shella) - سوق خدمات عربي

<div dir="rtl">

## نظرة عامة
تطبيق **شلة** هو سوق خدمات عربي متكامل يدعم RTL بالكامل، مبني بـ Flutter مع تصميم يتبع مبادئ Material Design المُكيّفة للعالم العربي.

## المميزات
- دعم RTL كامل مع خط Cairo العربي
- تدفق تسجيل دخول متكامل (رقم جوال + OTP)
- لوحة خدمات متنوعة (الماركت، المطاعم، الصيانة، الصيدلية، التعليم، القانون، الجمال)
- شاشة رئيسية مع بانر إعلاني وكروت الخدمات
- دعم وضعي المستخدم: متلقي الخدمة / مقدم الخدمة

## البنية التقنية
```
lib/
├── core/
│   ├── theme/          # نظام التصميم الموحد (ألوان، خطوط، مسافات)
│   ├── constants/      # الثوابت والمتغيرات
│   └── routing/        # go_router مع مسارات مسماة
├── features/
│   ├── auth/           # تدفق تسجيل الدخول
│   └── home/           # الشاشة الرئيسية والخدمات
└── shared/
    └── widgets/        # مكونات قابلة لإعادة الاستخدام
    ```

    ## المتطلبات
    - Flutter >= 3.0.0
    - Dart >= 3.0.0

    ## خطوات التشغيل
    ```bash
    # استنساخ المستودع
    git clone https://github.com/groupshella/c.shella.git
    cd c.shella

    # تنزيل الحزم
    flutter pub get

    # تشغيل التطبيق
    flutter run
    ```

    ## المكتبات المستخدمة
    | المكتبة | الغرض |
    |---------|-------|
    | `flutter_riverpod` | إدارة الحالة |
    | `go_router` | التنقل بين الشاشات |
    | `google_fonts` | خط Cairo العربي |
    | `smooth_page_indicator` | مؤشر البانر الإعلاني |
    | `pinput` | خانات OTP |

    ## الشاشات
    1. **Welcome / Challenge** - صفحة الترحيب
    2. **Enter Number** - إدخال رقم الجوال
    3. **OTP** - رمز التفعيل
    4. **Wrong OTP** - حالة خطأ OTP
    5. **Create Account** - إنشاء حساب جديد
    6. **Choose Status** - اختيار نوع المستخدم
    7. **Home** - الشاشة الرئيسية مع جميع الخدمات
